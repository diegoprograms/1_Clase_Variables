# Clase: Variables en Python (Notebook tipo Colab dentro de GitHub)

Este repositorio está diseñado para trabajar con **Jupyter Notebooks** desde el navegador y hacer **commit** de los avances.

## Ejecutar en GitHub Codespaces (recomendado)
1. Abre el repositorio en GitHub.
2. Clic en **Code** → **Codespaces** → **Create codespace on main**.
3. Abre `notebooks/01_variables_python.ipynb`.
4. Selecciona el **Kernel: Python 3** si lo solicita.
5. Ejecuta celdas, resuelve ejercicios y luego haz **commit** y **push**.

## Edición rápida (sin ejecutar): github.dev
- Presiona la tecla **.** (punto) dentro del repositorio para abrir VS Code Web.
- Para **ejecutar** celdas usa Codespaces.

## Entregable sugerido
- Completar los ejercicios al final del notebook.
- Commit con mensaje claro (ej: `Ejercicios variables completados`).
